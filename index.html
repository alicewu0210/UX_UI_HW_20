About Project

The Problem

Through stakeholder discussions, we identified that the SIJCC needs to enhance its fundraising efforts. We discussed leveraging the events hosted by SIJCC as a way to inspire local participation while supporting the mission of building a more connected Los Angeles.


The Solution

We believe that by streamlining event information, and improving the user experience, we could increase event attendance and, in turn, boost fundraising efforts. The final deliverables included a final design comp in Figma and UI assets: typeface, color scheme, for developer to handoff.

    • User Interviews

    From stakeholder interviews, we identified user pain points. We found that the SIJCC needs to enhance its fundraising efforts, as highlighted by stakeholders. Additionally, we learned that the current website has an overwhelming amount of disorganized text, poor use of space, and inconsistent design, all of which hinder user-friendliness and prevent busy individuals from easily finding events. By eliminating these pain points from the new design, we ended up with a more streamlined and intuitive user experience. The redesigned website features a clean layout with well-organized content, making it easier for users to navigate and find information quickly.



    Problem Statement



    "How might we redesign this platform to prioritize user preferences and interests, enabling seamless discovery of events that align with their lifestyle?"

    Usatbility Testing

    We created a fully-functional, high-fidelity prototype of the new flows using Figma. At the same time, we started recruiting subjects for the test who fit our criteria. We did 4 usability tests in the first round and 3 after iterating on the issues that we’ve identified:

    Issue 01:

    "The Nav bar could be easier to navigate. There are a lot of words there”.

    User testing revealed that the long navigation bar resulted in insufficient spacing between the texts, causing users to feel overwhelmed and frustrated. The cluttered appearance reduced readability and made it difficult for users to quickly find the information they needed. As a result, users expressed a desire for a more streamlined and organized layout, with ample spacing and clear visual hierarchy to improve their overall experience on the site.


    Solution 01:

    Organize smilier information into one categories.


    Before

    Remove “Buy Tickets” : Create more space for navigation bar

    Iteration 1

    Combine all programs into “Our programs”: streamlined information with a smaller navigation.

    Final
    Issue 02:

    "CTA button is confusing".

    User testing reflected that the "Register Now" button on the event page was confusing. It felt like it could have been an instant purchase button. However, out intention of the button was to encourage user to click on it and learn more about the event.




    Solution 02:

    Rename CTA button.


    Before

    We changed it to “Learn more” button in case people want to read more about the show before purchasing, aligning our goals and stakeholder's requirement: encourage visitors to learn about the events and participate it.
